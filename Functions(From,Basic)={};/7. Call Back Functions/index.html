Callback Functions in JavaScript

What is callback

A callback function that is passed to another function as an argument and is executed after the first function has completed. Callback are often used in asynchronous to handle the result or error of the operation.

Basic Example

Below is an example of a function that accepts another function as its argument and calls it as a callback : 

        
            function greetUser(name, callback) {
                console.log("Hello, + name")
                callback;
            }

            function sayGoodbye () {
                console.log("GoodBye!");
            }

            greetUser("Rahul", sayGoodbye);
            // Expected output :
            // Hello Rahul
            // GoodBye!

Callback in Asynchronous operations

Callbacks are often used in asynchronous functions, such as reading files or fetching data from a server. Here's an example using `setTimeout` function to simulate an asynchronous operation. 

        
            function fetchData(callback) {
                setTimeout(function(){
                    let data = "'Sample Data'";
                    callback(data);
                }, 3000);
            }
            
            fetchData (function(data) {
                console.log(data);
            });

            // Expected output (after 3 seconds );
            // Sample Data

Conclusion

Callback functions provide a way to manage asynchronous operation and handle results or errors. While they're powerful, it's essential to be cautious with nested callbacks to avoid the "callback hell" scenario. Modern JavaScript offers alternatives like Promises and async/await for better asynchronous code management.
